HOMEWORK HELPER AI - SETUP INSTRUCTIONS
========================================

QUICKSTART (Windows 10):
1. Double-click "setup_and_run.bat" - it will automatically install dependencies and launch the app!
2. Add your OpenRouter API key in the Settings panel when the app opens

MANUAL SETUP:

STEP 1: Get OpenRouter API Key
-------------------------------
1. Go to https://openrouter.ai/keys
2. Click "Create Key", give it a name
3. Copy the key (starts with "sk-or-v1-...")
4. You can either:
   - Add it in the Settings panel after launching the app (RECOMMENDED)
   - OR manually edit config.json and paste it between the quotes: "api_key": "YOUR_KEY_HERE"

STEP 2: Install Brave Browser
------------------------------
1. Download from: https://brave.com/download/
2. Install to default location: C:\Program Files\BraveSoftware\Brave-Browser\
3. Right-click Brave shortcut, select Properties
4. In the "Target" field, add: --remote-debugging-port=9222

   Full target should be:
   "C:\Program Files\BraveSoftware\Brave-Browser\Application\brave.exe" --remote-debugging-port=9222

5. Click Apply, then OK

STEP 3: Install Python Dependencies
------------------------------------
METHOD A (Automatic):
- Double-click "setup_and_run.bat" and it will install everything!

METHOD B (Manual):
1. Open Command Prompt in this folder
2. Run: pip install -r requirements.txt
3. Run: python ui.py

TROUBLESHOOTING:
----------------
- If capture fails: Ensure you're on an edmentum.com page before capturing
- If connection fails: Ensure Brave is running with --remote-debugging-port=9222
- If ChromeDriver error: The app will auto-detect ChromeDriver from your system PATH
